# Agent 4: Data Processor MCP Server

A Model Context Protocol server for processing and transforming data.

## Features

- **validate_data()** - Validate data and return statistics
- **transform_data()** - Flatten, pivot, aggregate, normalize
- **aggregate_data()** - Group and compute statistics
- **convert_format()** - Convert between formats
- **detect_anomalies()** - Find outliers and anomalies

## Installation

```bash
pip install -r requirements.txt
```

## Running

```bash
python src/mcp_server_data_processor/server.py
```
