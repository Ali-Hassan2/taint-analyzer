# Agent 3: Code Analyzer MCP Server

A Model Context Protocol server for analyzing code quality and security.

## Features

- **analyze_code_quality()** - Compute complexity and maintainability metrics
- **security_scan()** - Identify security vulnerabilities
- **review_code()** - Comprehensive code review
- **suggest_refactoring()** - Suggest refactoring opportunities

## Installation

```bash
pip install -r requirements.txt
```

## Running

```bash
python src/mcp_server_code_analyzer/server.py
```
